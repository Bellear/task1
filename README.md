Задание 1.
Для запуска добавить в php.ini мультибайтовые команды, запустить через консоль schet.php (php schet.php).

Переменная $text – исходный текст для анализа.
Переменная $word – слово, повторение которого нужно посчитать в тексте.

Задание 2.
index.php – страница с формами.
headling.php – файл обработки текста и файла, загруженных пользователем.
schet.php – функция обработки из первого задания немношк измененная.

Пользователь может ввести текст в форму и/или загрузить файл с текстом и указать слово для поиска в тексте.
Результатом выполнения будет сгенерированные .csv файлы (отдельно для текстового поля и загруженного файла) в которых содержится подсчет слов.
